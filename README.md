# SARA - Servicios e Ingeniería

Landing page moderna para Servicios e Ingeniería SARA, empresa mexicana especializada en mantenimiento industrial.

## Características

- ✅ Next.js 15.5.10
- ✅ React 18.3
- ✅ TypeScript 5.3
- ✅ Tailwind CSS 3.4
- ✅ React Icons 5.0
- ✅ Diseño responsive
- ✅ 0 vulnerabilidades

## Desarrollo

```bash
# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm run dev

# Construir para producción
npm run build

# Iniciar servidor de producción
npm start
```

## Secciones

- **Hero**: Presentación principal
- **About**: Quiénes somos, misión, visión y valores
- **Services**: Servicios de mantenimiento industrial
- **Brands**: Marcas con las que trabajamos
- **Contact**: Información de contacto
